# LinkedIn Profile Links Collector (Chrome Extension)

This Chrome extension allows you to **copy all LinkedIn personal profile links** (`/in/`) from a LinkedIn search results page with a single click.

It is useful for recruiters, sourcers, or anyone who wants to collect profile URLs from LinkedIn search results for later analysis, outreach, or organization.
